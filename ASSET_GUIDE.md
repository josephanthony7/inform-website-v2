## File Format Requirements

For the die-cut sticker aesthetic:
- **Format:** PNG
- **Background:** transparent when possible
- **Color Mode:** RGB
- **Optimization:** run through TinyPNG or ImageOptim

## How to Update Assets

1. Replace the file in the correct folder.
2. Keep the filename exactly the same.
3. Refresh the site to see updates.
